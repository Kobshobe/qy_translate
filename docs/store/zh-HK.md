### Summary in the software package

劃詞翻譯，接入大模型 API 翻譯，提供收藏管理及語言學習

### Explanation

劃詞翻譯擴充功能，接入 OpenAI、DeepSeek、Claude、Qwen、MiniMax 等多種大模型 API 進行翻譯，翻譯結果更智能、更準確、更自然流暢。支援自訂翻譯 prompt，可根據不同場景定制翻譯風格。

提供專業和簡約智慧兩種互動模式，滿足進階用戶和日常使用需求，可根據個人喜好自由配寘。

內寘 PDF 閱讀器，支援直接在瀏覽器內開啟和翻譯 PDF 檔案，無需額外工具即可輕鬆理解外文文檔。

支援夜間黑暗主題模式，夜間使用更護眼。

快捷鍵：mac 按下 Command+K，Windows 及其他系統按下 Alt+K，快速喚起翻譯視窗。

開源免費，更多使用說明請查看：https://github.com/Kobshobe/qy_translate
