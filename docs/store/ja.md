### Summary in the software package

選択したテキストを翻訳、大規模モデルAPI翻訳に対応、コレクション管理と言語学習を提供

### Explanation

大規模モデルAPI翻訳と統合され、よりインテリジェントで正確な翻訳を実現。OpenAI、DeepSeek、Claude、Qwen、MiniMaxなどのモデルに対応しています。

プロフェッショナルモードとシンプルモードの2種類のインタラクションモデルを提供し、ユーザーのニーズに合わせて設定できます。

PDF文書翻訳のためのPDFリーダーが内蔵されています。

ナイトダークテーマモードに対応。

ショートカットキー：Mac（Command + K）、Windowsその他（Alt + K）

注意：インストール後、開いているページを更新してください。

オープンソース、詳細：https://github.com/Kobshobe/qy_translate
