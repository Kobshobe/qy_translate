### Summary in the software package

划词翻译，接入大模型 API 翻译，提供收藏管理及语言学习

### Explanation

划词翻译扩展功能，接入 OpenAI、DeepSeek、Claude、Qwen、MiniMax 等多种大模型 API 进行翻译，翻译结果更智能、更准确、更自然流畅。支持自定义翻译 prompt，可根据不同场景定制翻译风格。

提供专业和简约智能两种交互模式，满足进阶用户和日常使用需求，可根据个人喜好自由配置。

内置 PDF 阅读器，支持直接在浏览器内打开和翻译 PDF 文档，无需额外工具即可轻松理解外语文档。

支持夜间黑暗主题模式，夜间使用更护眼。

快捷键：mac 按下 Command+K，Windows 及其他系统按下 Alt+K，快速唤起翻译窗口。

注意：安装后请刷新已打开的页面后使用
