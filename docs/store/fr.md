### Summary in the software package

Traduisez le texte sélectionné, intégré avec la traduction par API de grands modèles de langage, offrant la gestion des collections et l'apprentissage des langues

### Explanation

Intégré avec la traduction par API de grands modèles de langage, plus intelligent et précis, prend en charge OpenAI, DeepSeek, Claude, Qwen, MiniMax et d'autres modèles.

Offre deux modes d'interaction : professionnel et minimaliste intelligent, configurables selon les besoins.

Lecteur PDF intégré pour la traduction de documents PDF.

Prend en charge le mode sombre nocturne.

Raccourci : Mac (Commande + K), Windows et autres (Alt + K)

Remarque : veuillez actualiser la page ouverte après l'installation.

Open source, plus d'informations : https://github.com/Kobshobe/qy_translate
